HTML Basics

Text Formatting Elements

In HTML, we use specific tags to change the visual importance or style of text. While CSS is used for heavy styling, these HTML tags provide semantic meaning to your words.

Bold and Italicized Text

The most common formatting tags are used to emphasize specific words within a paragraph.

formatting.html
<p>This is <b>bold text</b> and this is <i>italic text</i>.</p> <p>Use <strong>important text</strong> for SEO importance.</p>

Semantic vs. Visual

There is a difference between <b> (visual bold) and <strong> (semantic importance). Screen readers for visually impaired users will emphasize strong text differently than just bold text.

more-tags.html
<p>You can also <mark>highlight</mark> text or show <del>deleted</del> content.</p>
