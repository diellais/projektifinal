HTML Basics

Understanding Elements & Tags

HTML is made up of elements. An element usually consists of a start tag, some content, and an end tag. Think of it like a container for your content.

example.html
<h1>This is a Heading</h1> <p>This is a paragraph.</p>

In the example above, the <h1> tag tells the browser that "This is a Heading" should be displayed as the main title of the page.
